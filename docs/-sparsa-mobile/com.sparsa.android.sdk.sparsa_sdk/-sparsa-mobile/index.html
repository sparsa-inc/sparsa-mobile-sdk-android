SparsaMobile

object SparsaMobile

Entry point for the Sparsa Mobile SDK.

This singleton provides access to all core SDK functionality, including initialization, device management, language settings, and credential operations. Ensure configure is called before using other methods.

The SDK supports various operations such as:

  • Device management (adding, removing, listing devices)

  • Credential management (retrieving, verifying credentials)

  • Language settings

  • Recovery email management

  • Device bootstrapping and verification

  • Notification handling
