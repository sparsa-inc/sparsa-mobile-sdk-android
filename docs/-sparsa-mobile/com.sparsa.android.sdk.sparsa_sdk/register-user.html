suspend fun SparsaMobile.registerUser(attributes: String): UserAuthenticationModel

Registers a new user and imports their digital address using a QR code or AppLink.

This method registers a new user by processing the data from a QR code or AppLink. It performs identity verification, including document and biometric verification, and creates a new digital address for the user.

The registration process involves:

  1. Parsing the attributes from the QR code or AppLink

  2. Retrieving KYC (Know Your Customer) validation settings

  3. Verifying the user's identity using the appropriate identificator

  4. Validating the user's information (SSN, biometrics, etc.)

  5. Registering the user with the KYC service

  6. Linking the current device to the new digital address

Return

UserAuthenticationModel A model containing the newly created digital address and the linked device ID.

Parameters

attributes

A string containing the data from the QR code or AppLink, typically in JSON format.

Throws

SparsaError.VerificationFailed

If identity verification fails or required data is missing.

SparsaError

If any other error occurs during the registration process.
