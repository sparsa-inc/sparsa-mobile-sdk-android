suspend fun SparsaMobile.authenticateUser(attributes: String): UserAuthenticationModel

Imports a digital address using the QR code or AppLink sent with a recovery email.

This method authenticates a user by processing the data from a QR code or AppLink that was sent to the user's recovery email. It performs biometric verification and imports the user's digital address to the current device.

The authentication process involves:

  1. Parsing the attributes from the QR code or AppLink

  2. Initializing the appropriate authenticator based on the environment

  3. Performing biometric authentication

  4. Processing the recovery and linking the device to the digital address

Return

UserAuthenticationModel A model containing the imported digital address and the linked device ID.

Parameters

attributes

A string containing the data from the QR code or AppLink, typically in JSON format.

Throws

SparsaError.AlreadyImported

If a digital address is already imported on this device.

SparsaError.SdkNotConfigured

If the SDK is not properly configured.

SparsaError.VerificationFailed

If the biometric verification fails.

SparsaError.RequestFailed

If the server request fails.
